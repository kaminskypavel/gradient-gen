# Gradient Gen

Infinite gradient wallpaper generator. Pick two colors, choose a shape, adjust sharpness, and download at up to 4K.

**[Live Demo](https://kaminskypavel.github.io/gradient-gen/)**

## Shapes

- **Waves** — flowing layered hills
- **Orbs** — overlapping spheres with 3D shading
- **Aurora** — vertical curtains of light
- **Mesh** — Voronoi-style smooth blending
- **Ribbons** — diagonal flowing bands
- **Crystals** — angular geometric facets

## Keyboard Shortcuts

- `Space` / `Enter` — generate
- `Cmd+S` — download

## Stack

React, Vite, Tailwind CSS v4, TypeScript. Zero runtime dependencies beyond React.

## Development

```bash
bun install
bun run dev
```
